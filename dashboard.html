ArcGIS Dashboards: City of Victoria Tree Species


    ArcGIS Dashboards is a web application for creating and sharing interactive dashboards that visualize and analyze data in real-time. Users can be customize dashboards with various widgets like maps, charts, and gauges, filtering and drilling down into the data to gain deeper insights.

    The Tree Species Dashboard was created using open data published by the City of Victoria, BC. This dashboard provides an overview of the tree species in the city, including their distribution and dimensions. It allows users to filter the data by species and location, providing a visual representation of the metrics as they interact with the map and the chart.
