Nanaimo Schools using Google Maps API

  • JavaScript functions are nifty little snippets that help you add additional functionality on to your map. In this map, I am using Google Maps JavaScript API and creating a map showing all schools within 1.5km of Nanaimo District Secondary School. I have included a search radius, scalebar, and geolocation option to find the current location.
  • The map also has a census tract population boundary for Nanaimo symbolized in orange color. This layer was loaded as Geo JSON file using Features to JSON geoprocessing tool on ArcGIS Pro.
  • The search radius symbolized in green was created using the Draw on map example available on the Google Developers website.
  • Next, I write a function to create markers with popups for each school in the search radius and formatted the popups to include an image of the school.
  • Finally, I write a snippet of code to set up geolocation with error handling using a button 'Pan to current location'. This code uses Event Listener to ask user whether the website can use the user's location. Based on the user's the choice, the map either pans to current location or shows an error message.
