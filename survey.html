ArcGIS Survey123 Web Designer & Survey123 Connect


    Survey123 lets you design and distribute web-based surveys, linking form questions to feature layer fields. You can customize question behavior and share the survey via a link or QR code. Responses are stored in the feature layer for easy access and analysis. Surveys can be created using two tools: Web Designer on the Survey123 website or Survey123 Connect software.
    Web Designer is a browser-based app with a drag-and-drop interface for quickly building surveys from scratch or templates. Survey123 Connect, installed on the desktop, uses an XLSForm-compliant spreadsheet, to create complex functionalities and custom designs. Check out the following 2 surveys:

  • Visitor Survey - Moose Sightings created using Web Designer: This survey collects text, date, time, and location data using basic functionalities of Survey123 Web Designer.

  • Restaurant Reservations created using Survey123 Connect desktop tool: This survey uses additional features from Survey123 Connect XLSForm, including groups, lists for single/multiple choice questions, dependencies based on previous answers, variable calculations, regex constraints, and diverse appearances for multi-choice questions.
