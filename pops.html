Choropleth Map of Population and Dwelling Counts 2021


  • Statistics Canada publishes population datasets every five years, this data is based on the Census of Population, 2021. To perform this analysis, we use two files: Boundary files by dissemination areas and Population and Dwelling counts csv. These files have a common identifier DGUID (statistical) and therefore, we can use Add Join function to combine these two datasets.
  • The first map is the Population filtered by Nanaimo census division and symbolized using Natural Breaks method, and raw population count i.e. number of individuals in each dissemination area.
  • The second map is the Population Density per sq km filtered by Nanaimo, symbolized using Natural Breaks method and calculated using the raw population count Normalized by land area i.e. number of individuals per unit land area.
  • Finally, we compare private dwellings count vs. population count using Bivariate symbolization to visualize areas with Low-Low, Low-High, High-Low, and High-High relationship.
