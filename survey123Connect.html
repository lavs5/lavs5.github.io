ArcGIS Survey123 Connect


    ArcGIS Survey123 enables the design and distribution of web-based surveys by linking form questions to feature layer fields. Users can customize question behavior, share surveys via links or QR codes, and store responses directly in the feature layer for seamless access and analysis. For more advanced survey functionalities, Survey123 Connect, a desktop application, allows users to create complex and feature-rich surveys using XLSForm-compliant spreadsheets. This approach supports customized designs, intricate logic, and enhanced user interaction.
