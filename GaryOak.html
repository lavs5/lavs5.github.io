Garry Oaks of Victoria using Folium



  • Maps showing Garry Oak/ Oregon White Oak tree species distribution in the City of Victoria
  • Folium is a library that uses JavaScript leaflet.js module in the background and it enables interactive map visualizations in Python.
  • In this example, I am analyzing tree species published by the City of Victoria's Open Data portal using Folium, folium plugins such as HeatMap and MarkerCluster, and geopandas Python libraries.
  • In the first step, I set up a Open Street Basemap with zoomed in view on the City of Victoria coordinates.
  • Next, we read a shapefile representing locations of Garry Oak tree species in the City of Victoria. Using a for loop on the geopandas dataframe, I am creating markers for each tree location.
  • However, this map is not easy to read, therefore, in the next step I use MarkerCluster to create clusters with number of trees in each cluster. Another form of symbolization is using Folium Circles of specified radius and color.
  • Finally, I am using the heatmap plugin to visualize the distribution of Garry Oak trees where brighter yellow/orange represent higher density.
