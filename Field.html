ArcGIS Field Map Designer


    ArcGIS Field Maps Designer is a web-based tool that enables users to configure and optimize maps for mobile field data collection. It streamlines field workflows by allowing users to customize forms, link attributes to feature layers, and enhance data collection processes. Users can set up required fields, apply conditional logic, and optimize mobile workflows for field staff.
